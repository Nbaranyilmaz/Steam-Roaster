# Steam Roaster

Steam Roaster is a desktop application designed to analyze Steam profiles and brutally Roasts gaming tastes and total playtimes. This AI-powered app offers sharp, unfiltered reviews based on your Steam library.

> **WARNING:** This application provides harsh criticism and may cause emotional damage.

## Features

Uses Google Gemini to generate unique, context-aware roasts based on library and playtime data.
It has a clean, dark-themed user interface that won't make your eyes bleed.

## Download and Usage (Windows)

No installation or coding knowledge is required to use the application.

1.  Navigate to the **Releases** section on the right side of this page.
2.  Download the latest `Steam Roaster.exe` file.
3.  Double-click the file to launch the application.
4.  Paste a Steam profile link, ID, or username into the input field.
5.  Click the **ROAST** (or YARGILA) button.
