## Features

3-There are 2 language options: English & Turkish.


## Download and Usage (Windows)
No installation or coding knowledge is required to use the application.

1.  Navigate to the **Releases** section on the right side of this page.
2.  Download the latest `Steam Roaster.exe` file.
3.  Double-click the file to launch the application.
4.  Paste a Steam profile link, ID, or username into the input field.
5.  Click the **ROAST** (or YARGILA) button.
